# my-study-playground
# Windows DFIR / Threat Hunting Study Playground

## 目的
Windows DFIR・Threat Hunting・PowerShell調査の実践力をつけるための学習ログ。
---

## 学習方針
- 1週間 = 1テーマ（攻撃手法）
- 深く掘る（広げない）
- ログ・挙動ベースで理解する
- ノートはGitに集約する
---

## 1週間の流れ

### 日曜
テーマ決定・準備・基礎調査

### 平日
調査・ノート追記・軽い検証

### 週末
ラボ・ログ解析・まとめ

---

## 各週の中身
- notes.md（まとめ）
- lab.md（検証）
- detection.md（検知観点）
- logs.md（ログ確認）
- cases.md（THM / LDなど）
- references.md（参考）

---

## ゴール
Windowsの攻撃を見たときに、
「どう調査するか」を自分で組み立てられる状態にする。
